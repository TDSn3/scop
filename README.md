# scop
Basic GPU rendering with Vulkan.

# Usage

### MacOS

GLFW configuration :

``` shell
mkdir ./library/glfw-3.4/build
cd ./library/glfw-3.4/build
cmake -DBUILD_SHARED_LIBS=OFF ..
make
```
